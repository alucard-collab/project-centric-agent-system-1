# Project-Centric Agent System (PCAS)

지능형 에이전트들이 TF팀을 구성하여 자율적으로 프로젝트를 수행하고 검증하는 차세대 멀티 에이전트 플랫폼입니다.

## 🌟 Core Concepts
- **Channel-Based**: 프로젝트별 독립된 세션 및 지식 관리.
- **Hierarchical Loop**: 주니어의 실행력과 시니어의 검증력이 결합된 품질 보증 루프.
- **Modular & Local-First**: 100% 로컬 LLM 연동 및 확장 가능한 모듈형 아키텍처.

## 🧠 Master Brain
본 프로젝트의 상세 설계 및 개발 로그는 사용자님의 **Master Brain**에서 통합 관리됩니다.

## 📂 Directory Structure
- `agents/`: 에이전트 페르소나 및 도구 정의
- `src/`: 시스템 핵심 로직 (Bridge, Storage, UI)
- `pcas/`: 로컬 세션 및 데이터 저장소
